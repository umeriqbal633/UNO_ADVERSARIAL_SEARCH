# 🃏 UNO AI Simulator: Adversarial Search

An interactive, graphical UNO simulation built in Python that pits a human player against advanced Artificial Intelligence agents. This project demonstrates the practical application of adversarial search algorithms (**Minimax** and **Expectimax**) with custom heuristic evaluation functions in a dynamic, multi-agent environment.

## ✨ Features

- **Human vs. AI Gameplay**: Fully interactive PyGame GUI with clickable cards, valid move highlights, and dynamic visual state states (Main Menu, End Game popups).
- **Minimax Agent (Balanced)**: Uses the Minimax algorithm with Alpha-Beta pruning to evaluate game trees. Geared towards a balanced approach, it minimizes opponents' advantages while efficiently managing its own hand.
- **Expectimax Agent (Offensive)**: Uses Expectimax to handle "chance nodes" (the unpredictability of drawing cards from the deck), making highly aggressive and calculated offensive moves.
- **Auto-Play Mode**: Watch the AI agents battle it out automatically at the press of a button.
- **Detailed Game Logs**: A live terminal log in the GUI that renders decision metric points, expected evaluations, and game states.

## 🧠 AI Architecture & Heuristics

Rather than making random legal moves, the AI players simulate future turns using a game tree. They evaluate the "goodness" of a simulated state using a custom heuristic function:

`Score = 50 − 5(C_AI) + 2(C_opp) + 3(S)`

- `C_AI`: The number of cards in the AI's hand (lower is better).
- `C_opp`: The number of cards in the opponent's hand.
- `S`: Special action modifiers (e.g., hoarding Draw 4s or Skips increases score).

## 🎮 Controls

- **Mouse Click**: Select your cards to play, or click the Deck to draw.
- **A**: Toggle "Auto-Play" mode (automatically plays turns for the AI without needing to prompt them).
- **Spacebar**: Proceed to the next AI turn step-by-step.
- **R**: Reset the game.
- **Esc**: Exit the simulator.
